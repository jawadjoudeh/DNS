Setup Guide

Quick Start

1
Get your API Key
Go to the API Keys page and copy your key.
2
Add the header
Include your key in the X-API-Key header of every request.
3
Call the API
Send POST requests to /api/v1/check with a JSON body.

Authentication

Include your API key in every request header:

X-API-Key: sk-your_api_key_here

POST /api/v1/check

Classify a domain against the ML model.

Request:

curl -X POST http://localhost:5000/api/v1/check \ -H "Content-Type: application/json" \ -H "X-API-Key: YOUR_KEY" \ -d '{"domain": "example.com"}'

Response:

{ "prediction": "Safe", "confidence": 0.97, "blocked": false, "action": "Allowed", "attack_type": null, "features": [11, 0, 7, 3.02, 0, 0.0, 0.36, 0.45, 0.09, 0] }

POST /api/classify

Classify a domain (session-authenticated, no API key needed if logged in).

POST /api/classify Content-Type: application/json {"domain": "suspicious-domain.xyz"}
